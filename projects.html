Snake Game

Snake Game (Java Project – First Year) As part of my first-year university coursework, I developed a basic Snake Game using Java. The game features classic gameplay where the player controls a snake that grows longer as it eats food, while avoiding collisions with the walls and itself. The project was built using Java's library for the graphical interface and basic object-oriented programming principles for game logic. This project helped me strengthen my understanding of Java, event handling, and game loop logic.

Fikar Not (Freelance Platform for Technically able Workers)

Fikar Not – Entrepreneurship Project. As part of my entrepreneurship project, I developed a business idea called "Fikar Not", a platform designed to connect people with skilled technical workers like plumbers, electricians, carpenters, and more. The service allows workers to submit their gigs and showcase their skills, while users can easily find and hire reliable professionals anytime without hassle. The goal was to bridge the gap between service providers and customers through a simple, accessible solution.
